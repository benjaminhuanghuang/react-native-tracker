# React Native Tracker
  A REST-ful API mobile client.

## Using
  react native
  
  redux

  react route

## Features
  Login, Logout
  
  View routing

  


## Start up
  npm install -g react-native-cli
  
  react-native init tracker
  
  cd 
  
  react-native run-ios 
  
  react-native run-android
